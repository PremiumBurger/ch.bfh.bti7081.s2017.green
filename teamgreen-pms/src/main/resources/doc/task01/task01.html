Task 01 - First analysis

In your team, identify

  1. target users
  2. key features (from the users' perspective)
  3. critical success factors

Target Users

Benutzer welche dieses System bewirtschaften werden
# Rolle Jobbezeichnung Zugriff
1 Medical Users Docotrs, Nurses, Healt visitors Vollzugriff (admin)
2 Other clinical staff Receptionist Data-Input
3 Hospital management ? No access (erhalten generierte Reports durch Receptionst's)

Key features

Hauptpunkte welche dafür sprechen dieses System zu bauen
Rolle Features
Medical Users
  • Medikamentierung - CRUD
  • Behandlungen - CRUD
  • System warnt falls Medikamente falsch dosiert verschrieben werden
Clinical Staff
  • Patientenverwaltung (Stammdatenpflege)
  • Patienten können als "gefährdet" gekennzeichnet werden
  • Auflisten von Ärzten / Kliniken welche für den Patienten X geeignet sind
  • Planen von Terminen - Patient X am dd.MM.yyyyT14:30:00 mit Arzt Y in Klinik Z
  • System verhindert Konsultationsüberschneidungen des selben Arztes
  • Reporte generieren (Anforderungen von Management)

  • Welche Software wird in der Praxis verwendet?

Tutoris (ähnlich wie ECase) Personalien werden erfasst, Beziehungen, Angehörige, verschieden Rubriken mit Statistiken. Falldokumentation und Beziehungen. Es gibt verschiedene Objekte(Statistik, Budget, Dokumente(Briefe, Vollmachten), Sozialversicherungen, Mietverträge, Zielvereinbarungen, Falldokumentation, Skonto).

Briefvorlagen

  • Abläufe?

Am Gespräch ist das System nicht verfügbar, Handnotizen werden im nachhinein im System erfasst.

Bei Falldokumentation, wird hinterlegt mit wem der Sozialarbeiter Kontakt hatte

  • Zentrale Datenbank?

Kein Zugriff von anderen Stellen, nur Sozialdienst Biel und alle Internen Stellen haben Zugriff (Mietfachstelle, Buchhaltung, Medizinisches Sekretariat, Arbeitsintegration)

  • Gibt es spezielle Prozesse für psychisch Kranke?

Keine Speziellen Prozesse nur in der individuellen Beratung wird der Klient an die entsprechenden Stelle weitergeleitet.

Adresse wird an Klient gegeben, Klient muss sich selber bei der entsprechenden Stelle melden.

Bei Gefährdung (Selbst oder Fremd-Gefährdung) wird eine Gefährdungsmeldung gemacht.

Die Meldung wird an die KESB gehen. (Schriftlich oder Telefonisch)

KESB ist eine Behörde in einer höheren Instanz. Mandat wird evt. Weitergegeben von KESB

Psychologie, Sozialarbeit, Ärzte, Anwälte.

  • Wie / Woher werden Patienten an diese Stelle verwiesen?

Sozialhilfe ist Subsidiär(alles andere muss zuerst überprüft werden), daher werden die Klienten nicht verwiesen, sie kommen freiwillig
